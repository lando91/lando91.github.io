B. Thực hành

1. Cấu hình thiết bị mạng không dây

Sinh viên thực hiện việc cấu hình thiết bị mạng không dây từ một mô hình mạng cho trước

Mô hình mạng dưới đây bao gồm các thiết bị định tuyến, chuyển mạch, thiết bị không dây và các thiết bị đầu cuối (các PC)

Sinh viên hãy thực hiện các bước sau đây để cấu hình cho phép PC3 kết nối vào hệ thống sẵn có thông qua kết nối với thiết bị không dây Wireless Router

Mở file Lab 5a.pka và làm theo hướng dẫn sau đây:

1.1 Kết nối thiết bị mạng không dây vào mô hình

Sử dụng cáp thẳng (Copper Straight-through) để kết nối từ cổng Internet của wireless router đến cổng Fa0/7 của switch

1.2 Cấu hình cơ bản cho Wireless Router

 ◆ Cấu hình phần Internet connection
Thực hiện các bước sau:
 - Bước 1: Chọn WRS2 > GUI tab
 - Bước 2: Chọn phần Internet Connection type thành Static IP
 - Bước 3: Cấu hình phần địa chỉ IP với thông số sau:
  • Internet IP address: 172.17.88.25
  • Subnet mask: 255.255.255.0
  • Default gateway: 172.17.88.1
 ◆ Cấu hình phần Network Setup
Thực hiện các bước sau:
 - Bước 1:Kéo xuống phần Network Setup
 - Bước 2: Điền Router IP là 172.17.40.1 và subnet mask là 255.255.255.0
 - Bước 3: Chọn Enabled cho phần DHCP server
 - Bước 4: Click Save Settings

1.3 Cấu hình truy cập và bảo mật cho Wireless Router

Thực hiện các bước sau:
 - Bước 1: Chọn tab Wireless ở phía trên cùng
 - Bước 2: Chuyển Network Mode thành Wireless-N Only và đổi tên của mạng không dây SSID thành WRS_LAN
 - Bước 3: Tắt SSID Broadcast (chọn Disabled) và click Save Settings
 - Bước 4: Chọn tab Wireless Security . Thay Security Mode từ Disabled thành WPA2 Personal. WEP, WPA, WPA2,…
 - Bước 5: Sử dụng mật khẩu cisco123 (phần passphrase)
 - Bước 6: Click Save Settings

1.4 Cấu hình Wireless Client

Thực hiện các bước sau:
 - Bước 1: Click PC3 > Desktop > PC Wireless
 - Bước 2: Click Profiles tab. Click New. Đăt tên Wireless Access
 - Bước 3: Trong màn hình tiếp theo, click Advanced Setup. Gõ WRS_LAN trong phần Wireless Network Name. Click Next
 - Bước 4: Chọn Obtain network settings automatically (DHCP) và click Next
 - Bước 5: Trong phần Wireless Security, chọn WPA2-Personal và click Next
 - Bước 6: Gõ passphrase là cisco123 và click Next
 - Bước 7: Click Save và click Connect to Network

1.5 Kiểm tra kết nối

Xem phần Signal Strength và Link Quality để biết thông tin link kết nối. Click More Information để xem chi tiết cấu hình địa chỉ IP

1.6 Kiểm tra kết quả thực hành

2. Cấu hình Router và PC

Sinh viên thực hiện việc cấu hình các thiết bị trong một mô hình mạng cho trước

Mô hình mạng dưới đây bao gồm các thiết bị định tuyến, các thiết bị chuyển mạch và các thiết bị đầu cuối (các PC)

Mô hình mạng thực hành Cấu hình địa chỉ IP trên router và PC

Bảng địa chỉ IP

Device Interface IP Address Subnet Mask Default Gateway
R1 G0/0 192.168.10.1 255.255.255.0 N/A
G0/1 192.168.11.1 255.255.255.0 N/A
S0/0/0 (DCE) 209.165.200.225 255.255.255.252 N/A
R2 G0/0 10.1.1.1 255.255.255.0 N/A
G0/1 10.1.2.1 255.255.255.0 N/A
S0/0/0 (DCE) 209.165.200.226 255.255.255.252 N/A
PC1 NIC 192.168.10.10 255.255.255.0 192.168.10.1
PC2 NIC 192.168.11.10 255.255.255.0 192.168.11.1
PC3 NIC 10.1.1.10 255.255.255.0 10.1.1.1
PC4 NIC 10.1.2.10 255.255.255.0 10.1.2.1

Sinh viên hãy thực hiện các bước sau đây để cấu hình cho phép các thiết bị trong mô hình kết nối với nhau

Mở file Lab 5b.pka và làm theo hướng dẫn sau đây:

2.1 Cấu hình địa chỉ IP cho các router

Trong mô hình mạng trên, Router 1, 2 chưa được cấu hình hoàn chỉnh. Vì vậy, tiến hành thực hiện cấu hình trên các Router

Thực hiện các bước sau:
 - Bước 1: Chọn R1CLI tab
 - Bước 2: Nhập password cisco để đăng nhập

User Access Verification
Password: cisco
R1>

 - Bước 3: Gõ lệnh enable và nhập password class

R1>enable
Password: class
R1#

 - Bước 4: Gõ lệnh config t để cấu hình các interface của Router 1

R1#config t
R1(config)#

 - Bước 5: Gõ các lệnh sau để cấu hình địa chỉ cho interface gigabitethernet 0/0 (địa chỉ được cấu hình là địa chỉ đã được xác định trong bảng địa chỉ)

R1(config)# interface gigabitethernet 0/0
R1(config-if)# ip address 192.168.10.1 255.255.255.0
R1(config-if)# no shutdown

 - Bước 6: Quan sát thông báo để biết đã cấu hình thành công Interface G0/0 hay chưa. Nếu đã thành công, gõ lệnh exit để thoát khỏi phần cấu hình của Interface G0/0

R1(config-if)# exit

 - Bước 7: Thực hiện lại Bước 5 để cấu hình địa chỉ cho các interface còn lại là G0/1 của R1, G0/0, G0/1 của R2 với các thông tin trong bảng địa chỉ  - Bước 8: Sau khi cấu hình xong, lưu lại cấu hình với câu lệnh copy run start

R1# copy run start
